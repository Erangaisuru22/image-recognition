# Image Recognition of Three Persons Using Machine Learning

## Overview

This project focuses on recognizing and classifying images of three different persons using Machine Learning techniques. The system is trained on facial images and can identify the correct person from a given image.

## Objectives

* Detect and recognize faces from images.
* Train machine learning models using facial image datasets.
* Compare the performance of multiple classification algorithms.
* Predict the identity of a person from a new image.

## Dataset

The dataset consists of facial images of three individuals:

* Person 1
* Person 2
* Person 3

Each person has multiple images captured under different lighting conditions and facial expressions.

## Machine Learning Algorithms

1. K-Nearest Neighbors (KNN)
2. Decision Tree
3. Random Forest
4. Support Vector Machine (SVM)

## Technologies Used

* Python
* Scikit-learn
* OpenCV
* NumPy
* Pandas
* Matplotlib

## Project Workflow

1. Collect facial images of three persons.
2. Preprocess images (resize, grayscale, normalization).
3. Extract facial features.
4. Split data into training and testing sets.
5. Train machine learning models.
6. Evaluate model performance.
7. Predict the identity of a person from a new image.

## Results

The models were evaluated using accuracy, precision, recall, and confusion matrices. Performance comparison was conducted to determine the most effective algorithm for person recognition.

## Future Improvements

* Increase the number of persons in the dataset.
* Implement deep learning models such as CNNs.
* Develop a real-time face recognition system using a webcam.
* Improve recognition accuracy under varying conditions.

## Conclusion

This project demonstrates the use of machine learning techniques for image-based person recognition. It provides a foundation for understanding image classification and facial recognition systems.
